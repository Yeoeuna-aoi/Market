¿Qué es Bootstrap y por qué se usa?
R:Bootstrap es un repositorio optimizado para sistemas móviles que facilita y agiliza el desarrollo web. Incluye diversos elementos 
diseñados con base en HTML, como fuentes, iconos, colores, botones, imágenes, formularios y bordes, además de una amplia variedad de librerías.
 Muchos desarrolladores utilizan Bootstrap para crear diseños predeterminados y maquetar sitios web de forma más rápida, aprovechando sus 
 componentes integrados. En lugar de crear estilos desde cero, se puede editar y adaptar lo que ofrece la librería, lo que permite agilizar 
 el desarrollo y optimizar el diseño de una página web.

¿Qué beneficios ofrece en diseño responsive?
R:Un diseño responsive es un formato de programación que permite que los sitios web se ajusten automáticamente 
a las características de los dispositivos de los usuarios, ya sean celulares, tablets o computadoras. 
Este tipo de diseño adapta la estructura, el tamaño y la disposición de los elementos para que la información se vea 
ordenada y priorizada, sin que se distorsione o se vea desorganizada. Además, aprovecha las funciones específicas de 
cada dispositivo, como la posibilidad de acercar o desplazarse fácilmente con el tacto en los móviles.

Diferencias entre usar solo CSS puro vs. CSS + Bootstrap
R:La diferencia entre usar CSS y CSS + Bootstrap es que CSS es una hoja de estilos usada en modo cascada donde se puede aplicar
 estilos al color, margen, fuente de párrafos, tablas, encabezados, etc. CSS permite cambiar la apariencia, el diseño y editar 
 estos tipos de diseños, aunque las propiedades de los estilos CSS son algo simples. En cambio, Bootstrap es una colección de clases 
 de CSS con funciones de JavaScript que incluye diseños responsivos, permitiendo maquetear más rápido la página web en el frontend 
 mediante librerías ya existentes, lo cual implica una estructura mucho más rápida con estilos CSS ya incluidos. Bootstrap incluye una 
 alta gama de componentes como barras de navegación, ventanas y alertas, elementos listos para usar que ahorran tiempo. 
 En CSS el desarrollador tiene el control total de la apariencia de la página web, pudiendo desarrollar estilos según la necesidad, 
 con facilidad para personalizar y crear diseños únicos y animaciones complejas, aunque aprender CSS desde cero es desafiante por la 
 gran cantidad de diseños y formas de desarrollarlo. Bootstrap es más adaptable al uso, permitiendo automatizar diferentes tamaños de 
 pantalla y reducir el tamaño necesario para adaptarse a dispositivos móviles y otros aparatos, garantizando una apariencia adecuada y 
 uniforme. Los diseños prediseñados de Bootstrap ayudan a diseñar de manera más rápida sin empezar desde cero, aligerando el desarrollo y 
 aumentando la productividad. Bootstrap tiene una comunidad muy activa y gran cantidad de documentación, permitiendo encontrar soluciones a 
 problemas comunes.
